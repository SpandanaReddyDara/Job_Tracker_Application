## Job Application Tracker:
This project is designed to help job seekers organize and streamline their job search process, with a strong focus on usability, emotional support, and future-of-work relevance.

## The app includes:
•	A simple login/signup page
•	Add Job form — with fields like Job Link, Title, Descriptoin, Category and Interview date.
•	A Dashboard — shows how many jobs per status, platform-wise breakdown
•	Interview alerts — automated reminders on the day of interviews

## Tech stack:
Built using React, Node.js and MongoDB
